系统资源相关命令

top

1、当前实时间，已开机时间，登陆用户数，系统负载
2、任务：总任务数，正在运行，休眠，停止，僵尸
3、cpu；用户使用，系统使用，改变过优先级的进程使用，空闲，等待，硬中断，软中断，表示本应该分配给虚拟机的资源被其他虚拟机或物理机本身占用而产生等待的比
4、总内存，已使用，空闲，写入硬盘的缓冲区
5、交换分区，总数，使用，空闲，cpu与内存的缓存

在top下按
1 #查看CPU核心数量
C #按照cpu使用率排序
M #按照内存使用率排序

系统负载

分别表示系统在过去1分钟、5分钟、15分钟内运行进程队列中的平均进程数量

进程的优先级(PR+NI):

越低越优先

PR（prioity）：优先级，用户不能控制的，默认20，RT（real time）实时
NI(nice值)：用户能够操作的优先级，范围-20~19,默认0
nice -n N command 将命令的nice值改为N
nice -n -20 ping 10.1.1.254
renice -n N PID根据进程修改nice值
