0、软件包组成

软件包组成：
不同类型的文件会装到不同的目录

可执行文件 #/bin /sbin/ usr/sbin..binary二进制
库文件dll，封装好的 #/lib /lib64   /usr/lib ...
配置文件    #/etc
帮助手册    #/usr/share/man

1、rpm(redhat packet manager)

后缀是.rpm

1.1、单个rpm包，rpm包命名

wireshark-1.8.10-17.el6.x86_64.rpm
软件名称-软件版本-修订号.使用的系统版本.硬件架构.rpm####使用rpm去管理软件

1.2、命令格式

rpm [options] [Package——Name]

1.3、查询

-q    #query    后面要加参数 软件名称
-qa    #all    查询所有安装过的rpm包
-qf    #file    -qf /usr/bin/xxd 找出这个文件是哪个包里面的，那就可以在别的服务器上安装某个特定的文件
-ql    #list，列表，查询rpm包包含哪些文件
-qi    #info，rpm -qi  vim-common查询软件包信息
--info            列出软件包的描述信息
-qc   查询指定程序安装生成的配置文件
-qd   查询指定安装包成成的文档
-q --scripts    
-p       查询针对是未安装的程序包文件
-qR    列出与软件有关的依赖软件所含的文件

1.4、导入证书

单个软件包存在依赖性
所有rpm包都会携带证书    软件和系统各有证书
导入证书 ls /etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-6
rpm --import /etc/pki/rpm-gpg/RPM-GPG-KEY-CentOS-6
把有依赖的包全部安装

1.5、安装

-ivh    
    i：install    v：verbose    h：hash
    i:install    安装
    v:verbose    显示安装过程信息
    h:hash    以#显示安装进度

实例：
rpm -ivh wireshark-1.8.10-17.el6.x86_64.rpm // --force强制重装

1.6、升级

rpm -Uvh    //man --help
rpm -Fvh    // 

注意：内核不能用-Uvh方式去升级而是用rpm -ivh方式去更新（uname、rpm -q kernel）
ivh方式可以在系统里面生成两个内核，可倒退

1.7、卸载

rpm -e +路径/直接软件名

1.8、附加

--oldpackage降级
--nodeps：当发生软件属性依赖问题而无法安装，可使用此参数强行安装
--replacefiles：覆盖文件
--replacepkgs：重新安装某个已安装的软件
--forc：强制安装
--test：检查软件的依赖性

1.9、校验：Verify

rpm {-V|--verify} [select-options] [verify-options]
参数：
-V： 后面接软件的名称，若该软件所含的文件被改动过，才会列出来
-Va：列出目前系统上所有可能被改动过的文件
-Vp：接文件名称，列出该软件可能改动的文件
-Vf：列出某个文件是否改动

2、yum：yellowdog updater modified

  • yum：是一个基于rpm的前端管理工具，解决依赖性

2.1、删除官方的仓库配置文件

rm /etc/yum.repos.d/*    //rm-remove

2.2、配置yum源

//仓库repository的配置文件
//创建仓库配置文件    vim    /etc/yum.repos.d/xx.repo
[local]    //仓库名称（不要漏掉中括号）
name=    //仓库描述
baseurl=http://mirrors.163.com/centos/6/os/x86_64    //仓库url
//baseurl=file：///media/cd
//baseurl=ftp：//10.1.1.251/cdrom repodata
mirrorlist
gpgkey：file：//etc/pki/rpm-gpg/RPM-GPG-KEY-Centos-6
gpgcheck=0 | 1//是否检查证书
enabled=0 | 1是否启用仓库（默认启用）

2.3清空yum缓存(当启用新的仓库时候)

yum clear all

2.4、查询

yum list      //查询仓库所有软件包名称
yum info     //查看软件包的信息--rpm -qa
yum list all  //
yum list installed //查看所有已经安装的软件包
yum --installroot==/ 指定安装目录
yum provides python    由文件查询安装包

2.5、安装

yum install    +软件名称就可以，不是rpm包的完整路径
yum reinstall    重装

2.6、 升级

yum update   #升级所有能够升级的软件包
yum update zsh   #升级指定软件包

2.7、卸载

yum remove

2.8、管理软件包组

2.8.1、查询
yum grouplist
yum groupinfo +包组

#####2.8.2、安装

yum groupinstall "X Window Systerm" "Desktop"
2.8.3、卸载
yum groupinstall "web server"
2.8.4、图形化相关包组：
X window System  //图形化底层的框架
Desktop 或者 KDE Desktop  //根据框架制作桌面（主题）
(Gnome,KDE,XFCE)
2.8.5、进入图形化：
    startx
    init 5
wim test
which yum
vim /usr/bin/yum
